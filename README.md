# The Path

スターウォーズ初心者向け視聴ルート提案Webアプリケーションです。

## 概要

The Path は、スターウォーズをこれから見始める人や、どの順番で見ればよいか迷っている人に向けた入門ナビゲーションサービスです。

ユーザーはアンケート診断やキャラクター検索を通して、自分に合った視聴ルートや作品を見つけることができます。

## 使用技術

- Python
- Django
- HTML
- CSS
- SQLite
- Git / GitHub

## 主な機能

- アンケート診断機能
- おすすめ視聴ルート表示機能
- キャラクター検索機能
- キャラクター詳細表示
- 作品一覧表示
- 作品詳細表示
## 制作目的

スターウォーズは映画、ドラマ、アニメなど作品数が多く、新規ファンにとって「何から見ればよいかわからない」という課題があります。

本サービスでは、ユーザーの興味や視聴経験に応じた視聴ルートを提案し、初心者でもスターウォーズの世界に入りやすくすることを目的としています。
